This is a simple API proxy to https://api.chkr.cc/ for checking credit cards.

How to Use

Make a GET request to the endpoint:

/api/check?cc=5121079814754898|05|2029|742

Example URL

https://chkr-api.vercel.app/api/check?cc=5121079814754898|05|2029|742

Notes

  • This API supports CORS and can be used from frontend apps directly.
  • If you access /api/check without cc query, you will get usage instructions in JSON.
